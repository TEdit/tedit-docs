# NPCs

The NPC panel lets you edit the custom names of town NPCs in your world.

![NPC Editor](/img/v5/sidebar-panels/sidebar-npc-editor.png)

Select an NPC from the list to view or change their display name. This is the name shown in-game above the NPC.

:::tip
To move NPC home positions, use the [Point tool](../interface/toolbar.md#point-tool-p) on the map.
:::
