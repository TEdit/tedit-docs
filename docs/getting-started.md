# Getting Started

:::danger
**Back up your world files before editing!**
:::

:::info
Make sure you've read the [Install Requirements](install-requirements.md) first.
:::

## Opening a World

1. Launch `TEdit.exe`
2. Use **File > Open** or `Ctrl+O` to open a `.wld` file
3. Terraria worlds are usually located at:
   - `Documents\My Games\Terraria\Worlds`

## Navigating the Map

| Action | Control |
|--------|---------|
| Pan | Middle-click + drag, or arrow keys |
| Zoom | Scroll wheel, or `Ctrl+=` / `Ctrl+-` |

:::tip
Zoom follows your cursor position — point at where you want to zoom into.
:::

## Your First Edit

1. Select the **Pencil** tool from the [toolbar](interface/toolbar.md)
2. Pick a tile type (e.g., Dirt Block) from the [action bar](interface/action-bar.md)
3. Left-click on the map to place tiles
4. Save with **File > Save** or `Ctrl+S`
