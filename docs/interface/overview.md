# Interface Overview

TEdit 5 uses a modern Fluent Design interface built on WPF-UI.

![TEdit 5 Interface](/img/v5/interface/TEdit_sNWvexY0S7.png)

## Layout

The editor is split into these main areas:

| Area | Location | Purpose |
|------|----------|---------|
| **Menu Bar** | Top | File, Edit, Layers, Plugins, Help |
| **Action Bar** | Below menu | Paint mode, brush settings, tile/wall selectors |
| **Toolbar** | Left side | Tool selector (Arrow, Pencil, Brush, etc.) |
| **Map View** | Center | The world canvas â€” where you edit |
| **Sidebar Panels** | Right side | Clipboard, world properties, find, filter, scripting |
| **Status Bar** | Bottom | Cursor coordinates, tile info, zoom level |

## Menu Bar

### File

![File Menu](/img/v5/interface/menu-file.png)

Open, save, and export worlds. Recent files list for quick access.

### Edit

![Edit Menu](/img/v5/interface/menu-edit.png)

Undo, redo, copy, paste, selection operations, and access to Settings.

### Layers

![Layers Menu](/img/v5/interface/menu-layers.png)

Toggle visibility of rendering layers (tiles, walls, wires, liquids, etc.).

### Plugins

![Plugins Menu](/img/v5/interface/menu-plugins.png)

Built-in plugin commands and utilities.

### Help

![Help Menu](/img/v5/interface/menu-help.png)

About, links, and update information.

## Dark Mode

TEdit 5 defaults to a dark theme. You can toggle light/dark mode in [Settings](../configuration/settings.md).
