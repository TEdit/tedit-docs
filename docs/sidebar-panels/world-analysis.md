# World Analysis

The World Analysis panel provides statistics about your world.

![World Analysis](/img/v5/sidebar-panels/sidebar-world-analysis.png)

## Features

- **Tile counts** — How many of each tile type exist
- **Biome breakdown** — Size and distribution of biomes
- **Chest inventory** — Summary of items across all chests
- **NPC status** — Which NPCs are present and where

:::tip
Use World Analysis with the [Find panel](find.md) to locate specific items or tiles you discover in the stats.
:::
