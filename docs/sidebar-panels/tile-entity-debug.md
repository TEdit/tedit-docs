# Tile Entity Debug

The Tile Entity Debug panel is an advanced tool for inspecting raw tile entity data.

![Tile Entity Debugger](/img/v5/sidebar-panels/sidebar-tile-entity-debugger.png)

This panel shows the internal data structure of tile entities (item frames, logic sensors, mannequins, etc.) and is primarily useful for debugging or advanced editing.
