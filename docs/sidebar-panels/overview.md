# Sidebar Panels

The sidebar panels are on the right side of the editor. Click the panel tabs to switch between them.

## Available Panels

| Panel | Purpose |
|-------|---------|
| [World Properties](world-properties.md) | World name, time, bosses, events, NPCs |
| [Special Tiles](special-tiles.md) | Edit chests, signs, tile entities |
| [Sprites](sprites.md) | Browse and place sprites/furniture |
| [Clipboard](clipboard.md) | Copy, paste, import/export schematics |
| [NPCs](npcs.md) | NPC name editor |
| [World Analysis](world-analysis.md) | World statistics and tile counts |
| [Bestiary](bestiary.md) | Bestiary kill counts and data |
| [Banners](banners.md) | Banner collection viewer |
| [Tile Entity Debug](tile-entity-debug.md) | Raw tile entity data inspector |
| [Creative Powers](creative-powers.md) | Journey Mode creative power settings |
| [Filter](filter.md) | Hide or grayscale tile layers |
| [Find](find.md) | Search your world for items, tiles, walls, sprites |
| [Scripting](scripting.md) | Write and run JavaScript/Lua scripts |
