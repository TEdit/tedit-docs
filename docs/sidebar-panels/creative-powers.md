# Creative Powers

The Creative Powers panel lets you edit Journey Mode settings stored in the world file.

If the world was created in Journey Mode, you can adjust creative power values like:

- **Time control** — Freeze or set time of day
- **Weather control** — Toggle rain, wind strength
- **Difficulty slider** — Enemy difficulty multiplier
- **Spawn rate** — Adjust enemy spawn rate
- **Infection spread** — Toggle biome spread

:::info
Creative Powers only apply to Journey Mode worlds. This panel has no effect on Classic, Expert, or Master mode worlds.
:::
