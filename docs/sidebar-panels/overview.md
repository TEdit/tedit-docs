# Sidebar Panels

The sidebar panels are on the right side of the editor. Click the panel tabs to switch between them.

## Available Panels

| Panel | Purpose |
|-------|---------|
| [Clipboard](clipboard.md) | Copy, paste, import/export schematics |
| [World Properties](world-properties.md) | World name, time, bosses, events, NPCs |
| [Special Tiles](special-tiles.md) | Edit chests, signs, item frames, mannequins |
| [Sprites](sprites.md) | Browse and place sprites/furniture |
| [Find](find.md) | Search your world for items, tiles, walls, sprites |
| [Filter](filter.md) | Hide or grayscale tile layers for clearer editing |
| [Scripting](scripting.md) | Write and run JavaScript/Lua scripts |
