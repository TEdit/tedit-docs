# World Properties

The World Properties panel lets you view and edit world metadata, progression flags, and NPC status.

![World Properties](/img/v5/sidebar-panels/sidebar-world-properties.png)

## Sections

### World Info

- **World Name** — Rename your world
- **Seed** — View the world seed
- **World ID** — Unique identifier
- **Surface / Bedrock Level** — Adjust terrain layer boundaries

### Time & Weather

- **Time** — Set in-game time of day
- **Moon Phase** — Cycle through moon phases
- **Blood Moon** / **Eclipse** — Toggle events
- **Rain** / **Sandstorm** — Toggle weather

### Boss Progression

Toggle which bosses have been defeated. This affects NPC spawning, world drops, and hardmode status.

![Boss Flags](/img/v5/sidebar-panels/sidebar-worldproperties-scroll1.png)

### Invasions & Events

- **Goblin Invasion** / **Pirate Invasion** / **Martian Madness**
- **Meteor spawned**
- **Smashed Orbs/Hearts** count

### NPCs

Manage town NPCs — add, remove, and set their home positions.

![NPC Details](/img/v5/sidebar-panels/sidebar-worldproperties-scroll3.png)

:::tip
Use the [Point tool](../interface/toolbar.md#point-tool-p) to visually place NPC homes on the map.
:::
