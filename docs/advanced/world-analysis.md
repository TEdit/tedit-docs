# World Analysis

TEdit 5 includes a world analysis panel that provides statistics and insights about your world.

![World Analysis](/img/v5/sidebar-panels/sidebar-world-analysis.png)

## Features

- **Tile counts** — How many of each tile type exist in the world
- **Biome breakdown** — Size and distribution of biomes
- **Chest inventory overview** — Summary of items across all chests
- **NPC status** — Which NPCs are present and their locations

:::tip
Combine World Analysis with the [Find panel](../sidebar-panels/find.md) to locate specific items or tiles you discover in the analysis.
:::
